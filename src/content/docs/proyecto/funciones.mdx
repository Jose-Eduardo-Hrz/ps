## Clonar una cadenas de caracteres

### Declaraci贸n
```c 
//  Biblioteca String.h
char * strdup( const char * string ) 
```

### Ejemplo
```c 
// archivo.c
#include <stdio.h>
#include<string.h>

int main() {
    char cadena [] = "Hola como estas!!!";
    char * p = strdup( cadena );
    printf( "%s" , p );
    return 0;
}
```

```sh
"Hola como estas!!!"
```
