## Problema 1

### Descripción 

Dado un valor entero "n" Construir una figura triangular, empleando ciclos.

### Ejemplo 

| Entrada | Salida |
|------|:-------|
7  | 7 6 5 4 3 2 1 <br/> 6 5 4 3 2 1 <br/> 5 4 3 2 1 <br/> 4 3 2 1 <br/> 3 2 1 <br/> 2 1 <br/> 2 1 <br/> 1 <br/>

### Código

```c 
#include<stdio.h>

void generarRenglones( int columnas ){
    for( int j = columnas ; j > 0 ; j-- ){
        printf( "%d " , j );
    }
    printf("\n");
}

void generarTriangulo( int n ){
    for( int i = 0 ; i < n ; i++ ){
        generarRenglones( n - i );
    }
}

int main() {
    int numero;
    scanf( "%d" , &numero );
    generarTriangulo( numero );
}
```

## Problema 2

### Descripción 

Dado un número entero "n" generado aleatoriamente, en el rango [1..500] imprimir en la pantalla todos los números primos que existen en el rango [1..n].

### Ejemplo 

| Entrada | Salida |
|------|:-------|
   <br/>    | 2 3 5 7

:::note[Nota]
La salida puede variar, ya que se utiliza numeros aleatorio, pero, solo debe mostrar numeros primos
:::

### Código

```c 
#include<stdio.h>
#include <stdlib.h>
#include <time.h>

int esPrimo( int numero ){
    for( int j = 2 ; j < numero ; j++ ){
        if( numero % j == 0 ) return 0;
    }
    return 1;
}

void imprimirNumerosPrimos( int numero ){
    for( int i = 2 ; i <= numero  ; i++ ){
        if( esPrimo( i ) ) printf( "%d " , i );
    }
}

int main() {
    srand(time(NULL));
    int numero = numero=rand() % 500 + 1;
    imprimirNumerosPrimos( numero );
}
```

:::note[Nota]
Para obtener numeros aleatorios en un ranfo de [ M ... N ] se utiliza la siguinete formula en la funcion `rand`:
```c 
rand() % (N - M + 1) M;
```
:::

## Problema 3

### Descripción 

Dado un número n, proporcionado por el usuario, la computadora deberá obtener la serie factorial de ese número así como el resultado total del factorial.  Investigar cómo se genera la serie de factorial.

### Ejemplo 

| Entrada | Salida |
|------|:-------|
5 | 5! = 5 X 4 X 3 X 2 X 1 = 120

### Código

```c 
#include<stdio.h>

void mostrarFactorial( int numero ){
    int resultado = 1;
    printf( "%d! = " , numero );
    for( int i = numero ; i > 1 ; i-- ){
        resultado = resultado * i;
        printf( "%d X " , i );
    }
    printf( "1 = %d" , resultado );
}

int main() {
    int numero;
    scanf( "%d" , &numero );
    mostrarFactorial( numero );
}
```
